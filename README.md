# KitchenFlo

KitchenFlo is a cloud-native restaurant operating system and POS platform.

## Features

- Restaurant POS
- Order Management
- Kitchen Display System
- Billing
- Inventory
- Multi-tenant SaaS architecture
- Procurement integrations
- Analytics
- Future AI integrations

## Tech Stack

### Frontend
- Next.js
- React
- Tailwind

### Backend
- Node.js
- TypeScript
- Express.js
- Prisma
- PostgreSQL
